Block-level Elements Example

This is an example of block-level elements:

  • <div>: Defines a division or section in an HTML document.
  • <h1>: Defines a top-level heading.
  • <p>: Defines a paragraph.
  • <ul>: Defines an unordered list.
    • <li>: Defines a list item.
    • Can contain other block-level or inline elements.

Inline Elements Example

This is an example of inline elements:

<span>: A generic inline container. <a>: Defines a hyperlink. <strong>: Indicates strong importance. <em>: Emphasizes text.
